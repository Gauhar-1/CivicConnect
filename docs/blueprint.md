# **App Name**: Civic Connect

## Core Features:

- Election Timeline: Displays upcoming elections in a clear, chronological order, allowing users to quickly view key dates, deadlines, and events.
- Manifesto Summaries: Generates AI-powered summaries of candidate manifestos. Integrates a large language model (LLM) to extract key points and convert them into concise overviews. The LLM acts as a summarization tool to Helps voters understand complex manifestos quickly.
- Candidate Directory: Searchable directory of candidates with individual profile cards. Each card shows name, party, region, key policies, and a link to full profile. Region-based filtering for ease of access.
- Live Feed: Real-time feed showing posts and announcements from candidates. Like, comment, share, and report options. Center-aligned LinkedIn-style feed; includes candidate badge and role indicators.
- Reporting System: Allows users to report election-related issues, candidate behavior, or false information. Attach photos/videos. Option for anonymous or verified reporting. Track report status.
- Campaign Discovery: Lets users discover and follow campaigns based on location, political party and popularity. Advanced sorting (e.g., top-rated, new, trending). Right sidebar widget + dedicated full-page directory.

## Style Guidelines:

- Primary Color: Navy Blue (#3B5998) Meaning: Trust, civic responsibility, and stability.
- Background Color: Light Blue (#EBF0F5) Usage: Clean backdrop that doesn’t compete with content; improves readability.
- Accent Color: Purple (#800080) Usage: CTA buttons, highlights, and alerts to draw user attention without clashing.
- Typefaces: Clean, professional, and readable fonts like `Inter`, `Roboto`, or `Open Sans`. Accessibility: Maintain minimum 16px for paragraph text, clear heading hierarchy, and strong contrast ratios.
- Main Layout Style: Inspired by LinkedIn for familiarity and usability. Left Sidebar: Navigation (Home, Directory, Timeline, Report, etc.) Center Panel: Content feed (posts, manifestos, campaign discovery). Right Sidebar: Trending elections, quick filters, volunteer signup prompts.
- Icons: Simple, universally recognizable icons (from libraries like `Lucide`, `Heroicons`, or `Material Icons`).
- Animations: Subtle and informative, e.g.: Button ripple effects, Feed loading skeletons, Transition animations on opening candidate profiles or reports.
